# 🍽️ WasteNot - Restaurant Food Waste Management

**WasteNot** is a Django-based platform that helps restaurants and cafes reduce food waste by tracking, managing, and redistributing surplus food. The platform enables businesses to donate excess food to charities or sell it at discounted prices to consumers — promoting sustainability and reducing environmental impact.

---

## 🌟 Features

- ✅ Track surplus food and monitor usage in real-time
- 📊 Smart analytics to reduce waste and cut costs
- 🏥 Donation management for charities
- 🛒 Discounted food sales to consumers
- 🔔 Alerts for soon-to-expire items
- 🌍 Environmental impact metrics
- 🔐 OAuth2 + JWT authentication system
- 💳 Razorpay integration for transactions

---

## 🛠️ Tech Stack

- **Backend**: Django, Django REST Framework (DRF)
- **Authentication**: OAuth 2.0, JWT
- **Task Queue**: Celery
- **Broker**: Redis
- **Database**: SQLite (default) / PostgreSQL (recommended)
- **Payments**: Razorpay
- **Containerization**: Docker, Docker Compose
